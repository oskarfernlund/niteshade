Installation
============

niteshade requires Python 3.6 or higher.

niteshade binaries may be installed from PyPI using pip 
https://pypi.org/project/niteshade/.

.. code-block:: console

    $ pip install niteshade

Use of a virtual environment is *strongly* recommended. 

.. note::
   Installation with conda is not currently available. Anaconda users should 
   install niteshade with pip inside an Anaconda environment.
